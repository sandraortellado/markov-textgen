How it works


via Gfycat

This web app uses a Markov chain to generate text. A Markov chain essentially consists of a set of transitions, which are determined by some probability distribution, that satisfy the Markov property: given a sequence, the next token in the sequence relies only on the current token and not on previous token. This means that given a string of words comprising a text, the next word in the sequence can be calculated only using a smaller set window of previous words to determine the next word.
